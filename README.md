# SonicHatch Landing Page

This repository contains the initial landing page for SonicHatch, a service offering professional home studio recording and mixing at an accessible price point.

## Features
- Responsive single-page layout with modern gradient branding.
- Booking waitlist form integrated with SonicHatch's public database for Service Inquiries.
- Clear presentation of service tiers with pricing.
- Contact section with professional email link.

## Technology
- HTML5, CSS (internal stylesheet)
- Google Fonts: Outfit (headings), Plus Jakarta Sans (body)
- JavaScript for form submission with async fetch to Baget public API

## Deployment
Deployed to Vercel via Baget deployment pipeline.
